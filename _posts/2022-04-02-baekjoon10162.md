## 코드

```python
T = int(input())

sec = [300, 60, 10]

cnt_list = []
for s in sec:   
    
    cnt = 0
    while T >= s:
        T -= s
        cnt +=1
        
    cnt_list.append(cnt)
    
if T != 0:
    print(-1)
else:
    for c in cnt_list:
        print(c, end = ' ')
```



## 문제 해설

간단한 그리디 알고리즘 문제. 출력하는 방식에 있어서 조금 헷갈릴 수도 있으나, T의 값이 0으로 알맞게 떨어지지 않는 경우에 -1을 출력하게 구현하면 간단히 해결할 수 있다.
