## 코드

```python
n = int(input())

for _ in range(n):
    bracket = input()
    stack = []

    for i in bracket:
        if i == '(':
            stack.append(i)
        elif (i == ')' and stack and stack[-1] == '('):
            stack.pop()
        else:
            stack.append(0)
            break
    
    print('NO' if stack else 'YES')
```
